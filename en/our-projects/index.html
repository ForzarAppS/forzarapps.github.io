Open source demo project «Check Update – Проверка Обновления»

Платформа:

Version: 1.0

Date of last update: 01.04.2020

Description: The program (free of charge, without using a server) shows how the check for updates for programs on Windows and Android platforms works using a text file on Yandex Disk. This program is written for informational purposes only for the concept of a verification algorithm. If necessary, the program can be rewritten into any programming language that is capable of performing these tasks.

Features:
  • Demonstration of the update check operation.
  • Free check for updates.
  • Open source.
  • Availability of automatic update check.

Recommendations:
  • More details about operators, procedures and functions you can find in the program help or on the site PascalABC.NET
    .
  • It is best to enable line numbering in the program settings.
  • You can debug the program through writeln (to the console) in places where you need to find out the value of a variable, and you will find out how the program is executed.
  • It is best to separate code fragments with empty lines.
  • For each new nesting of code, you must increase the indent on the left.
  • Be sure to comment out all the code.
  • Books on the programming system you can find at site PascalABC.NET
    .
