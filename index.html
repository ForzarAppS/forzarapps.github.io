Программа «Просмотрщик Шрифтов – Fonts Viewer»

Платформа:

Версия: 1.2

Дата последнего обновления: 22.03.2020

Описание: Программа подойдёт людям, которые всегда хотели иметь свой шрифт. А так же для людей, которым необходимо правильно расположить текст на клетчатом поле для дальнейшего его использования.

Особенности:
  • Создание и редактирование букв и символов любой сложности.
  • Просмотр слов и словосочетаний, состоящих из букв и символов ранее созданных шрифтов.
  • Позиционирование низа букв и символов.
  • Включение и отключения сетки размером 10х10 клеток.
  • Бегущая строка.
  • Наличие автоматической проверки обновления.

Демонстрационный проект с открытым исходным кодом «Проверка Обновления – Check Update»

Платформа:

Версия: 1.0

Дата последнего обновления: 01.04.2020

Описание: Программа (бесплатно, без использования сервера) показывает принцип работы проверки на наличие обновления у программ на платформах Windows и Android с использованием текстового файла на Я.Диске. Данная программа написана лишь в ознакомительных целях для понятия алгоритма проверки. При необходимости программу можно переписать на любой язык программирования, который способен выполнить эти задачи.

Особенности:
  • Демонстрация работы проверки обновления.
  • Бесплатная проверка обновлений.
  • Открытый исходный код.
  • Наличие автоматической проверки обновления.
